定义一个函数，让它输出你的名字，年龄，住址以及爱好，分四行打印出来。然后，调用这个函数。（解密点20）

def print_cv(name,age,address,hobby):
    print(name)
    print(age)
    print(address)
    print(hobby)
    
print_cv("栾生","40","南京路106号","打羽毛球")
