The first package: visPedigree 0.1

Sep 24, 2018 2 min read 可视化

今天对我来说，特别有意义，既是中秋节，也是我的第一个包visPedigree 0.1版本定型的日子。

数量遗传学基本上都是数字和公式，因此结果的可视化呈现变得非常重要。在windows平台下，有一个可用于显示个体系谱的pedigraph包，可以显示很多个体的系谱，功能非常强大，但是配置使用起来，需要摸索。在R环境下，kinship2, synbreed等包都可以画图，但是一旦系谱包括的个体数量多，画出来的系谱树，个体间重叠非常厉害。因此，开发了visPedigree包，可以用于显示很多个体(每个个体数>10000)的系谱。

  1. visPedigree包安装

目前visPedigree包还没有发布到cran，但是利用devtools包，已经可以从github上进行安装。

以rstrudio平台为例，如果没有安装devtools包，首先进行安装：

install.packages("devtools")

安装后，加载devtools

suppressPackageStartupMessages(require(devtools))

从github上安装visPedigree

install_github("luansheng/visPedigree")

然后加载visPedigree包。该包基于data.table和igraph包开发，如果这两个包没有安装，也会一并安装。

suppressPackageStartupMessages(require(visPedigree))

测试一下是否安装成功，显示visPedigree包自带的一个简单系谱:

head(simple_ped)

suppressWarnings(simple_ped_tidyped <- tidyped(simple_ped))

通常在Rstudio的Plots面板上显示的图片，清晰度较差，如果个体数量多，相互间也会发生重叠。将系谱图存为pdf文件可以有效的避免上述问题，获得高质量的矢量图。

suppressMessages(visped(simple_ped_tidyped, showgraph = FALSE, file="simpleped.pdf"))

打开simpleped.pdf文件，会看到高清晰度的系谱。
